# Projeto-3-Deploy do Stack de Infra de Dados no Azure e Terraform

> **Resumo:** Este projeto é uma implementação direta de Infraestrutura como Código (IaC) no Azure utilizando Terraform para provisionar Grupos de Recursos, Virtual Network, Subnets, Network Interface e Máquinas Virtuais. O ambiente integra um Dockerfile para a execução de aplicações em contêiner Docker e utiliza a ferramenta terraform graph para gerar o mapeamento visual da arquitetura.
